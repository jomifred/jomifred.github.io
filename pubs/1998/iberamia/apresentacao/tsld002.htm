Context

  • In a multi-agent system, the agents need to cooperate in order to achieve common goals

  • Co-operation implies having knowledge about others (their skills, desires, plans, etc), in such a way that one agent can request some task of another. Some of this information may be inferred from the social role the agent plays

  • For example: in a society of producers and consumers

    • The agents’ roles are: producers and consumers
    • From these roles, we can infer that an agent who is a producer has producing skills, wants to sell products, has a producer timetable, and so on

  • Knowing the roles of others is thus useful in both interaction and co-operation
