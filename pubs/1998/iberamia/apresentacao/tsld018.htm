Role identification from observation

  • Let y be the observer agent, x the observed one, z are others, and B a description of one role

    i) for each zi that x has had interaction with, one trace is constructed

    ii) for each trace an interaction process is constructed as described above, all these process form the set A

    iii) if there is one role model B that is subset of A, then x role is B
