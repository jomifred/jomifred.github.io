Identification of recursion

  • rewrite rule 3

      s ? I ; C.

      C = (P1 | P2 | P3 …) ; C.
    • I is a process that the object did at the beginning and didn’t do anymore
    • C is a cyclic process formed with alternatives process (Pi)
    • One Pi is identified if their correspondent trace is observed two or more times in s.
