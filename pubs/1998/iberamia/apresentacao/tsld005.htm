Social problems in agent migration

  • The main problem that a society must solve when an agent enters or leaves is how to maintain its functional integrity (Rocha Costa, 94)

    • After an agent migrates, the society must insure that it continues to function as well as (or better than) it did before the migration. The society left behind must be able to find among its remaining agents some that can assume responsibility for the function of the agent that left
    • The society the agent enters must be able to find ways to appropriately place the agent within its existing social structure and must also be able to modify that social structure, to enable the agent to perform its required functions.

  • A social structure of role links might be used to maintain functional integrity
