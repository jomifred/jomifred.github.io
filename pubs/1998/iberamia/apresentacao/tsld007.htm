Objective

  • The objective of this work is to specify a method for role identification from the observed action of another agent

    • This method can be used by an agent when it arrives in a new society and needs to know others’ roles in order to begin interaction

  • Some question ought to be answered in order to achieve this goal:

    • How do we describe and represent a role?
    • From a list of observed actions, how do we match this list with role description?
