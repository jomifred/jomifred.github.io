Process’ trace

  • “A trace of the behaviour of a process is a finite sequence of symbols recording the events in which the process has engaged up to some moment in time”

  • For example, the consumer trace could be

    • <>
    • <request product, receive, consume>
    • <request product, receive, consume, request product, receive, consume, request product, receive, return, request product, receive, consume>
