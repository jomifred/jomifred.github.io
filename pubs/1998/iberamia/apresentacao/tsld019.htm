A case study Step i) trace selection

  • Let the following actions observed from an agent named Renata:

    receive message(request(product),Jomi),

    receive message(request(space),Rocha),

    send message(reply(space_ok),Rocha),

    receive message(inform(product(_)),Rocha),

    send message(reply(product(_)),Jomi),

    receive message(request(space),Rocha),

    send message(reply(space_ok),Rocha),

    receive message(inform(product(_)),Rocha),

    receive message(request(product),Jomi),

    send message(reply(product(_)),Jomi),

    receive message(request(product),Jomi)

  • We have two traces:

    • one related to agent Rocha

    <receive message(request(space)),

    send message(reply(space_ok)),

    receive message(inform(product(_))),

    receive message(request(space)),

    send message(reply(space_ok)),

    receive message(inform(product(_)))>

    • another related to agent Jomi

    <receive message(request(product)),

    send message(reply(product(_))),

    receive message(request(product)),

    send message(reply(product(_))),

    receive message(request(product))>
