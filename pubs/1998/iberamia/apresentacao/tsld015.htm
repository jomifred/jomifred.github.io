Identification of choice

  • If two cyclic process Pi have the same beginning then we can conclude that there is an choice of behaviour.

  • For the last example, both processes begin equally

      P1 = (request product ? receive ? consume ? SKIP)

      P2 = (request product ? receive ? return ? SKIP)

  • So, we may create P like

      P = (request product ? receive ? (consume ? SKIP | return ? SKIP))

  • And the final identified process is

      Process = I ; C

      C = P ; C
